# 打牌记账小程序

一个使用 UniApp 框架开发的轻量级打牌记账小程序，支持多人游戏得分记录和历史查询。

## 功能特点

- **玩家管理**：支持添加、编辑、删除玩家（最多8人）
- **记账功能**：记录每局得分，自动计算总分
- **历史记录**：查看所有对局记录，按时间倒序显示
- **数据持久化**：使用本地存储，重启后数据不丢失
- **重置功能**：支持清空所有数据

## 使用方法

### 环境要求
- HBuilderX
- 微信开发者工具

### 运行步骤

1. 使用 HBuilderX 打开项目
2. 配置微信小程序 AppID（在 manifest.json 中）
3. 点击"运行" > "运行到小程序模拟器" > "微信开发者工具"
4. 或者在 HBuilderX 中发布为微信小程序

### 页面结构

- **首页**：显示所有玩家的当前总分排名
- **记账页**：输入本局各玩家得分
- **历史页**：查看历史对局记录
- **玩家管理**：添加、编辑、删除玩家

## 数据说明

- 玩家数据存储在 `card_players` 键下
- 游戏历史存储在 `card_game_history` 键下
- 所有数据使用 uni.setStorageSync 进行本地存储

## 技术栈

- UniApp 框架
- Vue 3
- 微信小程序

## 注意事项

1. 首次使用需要先添加玩家
2. 记账页面支持快捷加减分按钮（±10）
3. 支持正负整数得分输入
4. 清空数据操作不可恢复，请谨慎操作
